how does javascript work?

Javascript is a single threaded programming language and client side scripting language.single threaded means it has a single Call Stack. need an engine in the browser to run JavaScript.When the browser receives the code that we write, it follows the ECMA script rules for converted to machine code.Human readable code converts it into machine code in JavaScript engine via Just-in-Time Compiler, then after seeing machine code it on the computer, we see it as output in borowser

what does javascript event loop do?

We know that JavaScript is a single threaded programming language. When the call stack receives one of these asynchronous functions, it transmits the asynchronous function to the web API. when asynchronous execute and web api transfar a function to call back queue. when call stack all function executded done.Either exits the call back queue through the event loop and exits the code by bringing the function from the call stack. So it can be said that the function of the event loop is to bring the function from the callback queue into the call stack.asynchronous example: setTimeout(), setInterval(), fetch() etc..

what is difference between local storage and session storage?

Local storage and session storage do the same thing. There are a few differences between them.As long as a website is running and when the web site session is over the session storage data is expire.On the other hand, any data set in local storage data does not expire.Local storage data has to be removed explicitly.
